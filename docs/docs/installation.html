Before You Begin

Before you install Lookyloo, you should be familiar with the following:

*Lookyloo Concepts *Components installation *Cookies settings

Dependencies

Lookyloo requires the following to install:

  • Poetry: A tool to handle dependency installation as well as building and packaging of Python packages.

  • Splash: A javascript rendering service with an HTTP API. It’s a lightweight browser with an HTTP API, implemented in Python 3 using Twisted and QT5.

  • Redis: An open source (BSD licensed), in-memory data structure store, used as a database, cache and message broker.
